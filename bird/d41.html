दीघात समीकरण 4.1




जाँच कीजिये की निम्न दिघात समीकरण है|

(a) (x + 1)2 = 2(x-3)
(b) x2-2x = (-2)(3-x)
(c) (x-2)(x+1) = (x-1)(x+3)
(d) (x-3)(2x+1) = x(x + 5)
(e) (2x-1)(x-3) = (x + 5)(x - 1)
(f) x2+3x+1 = (x-2)2
(g) (x + 2)3 = 2x(x2-1)
(h) x3-4x2-x+1 = (x-2)3




(a) (x + 1)2 = 2(x-3)

=> x2+2x+1 = 2x-6

=> x2+ 2x+1-2x+6 = 0
=> x2+7 = 0
स्पष्ट x2+7 एक दिघात बहुपद है| अत: दी गई समीकरण एक दिघात समीकरण है|
(b) x2-2x = (-2)(3-x)

=> x2-2x+2(3-x) = 0
=> x2-2x+6-2x = 0
=> x2-4x+6 = 0
स्पष्ट x2-4x+6 एक दिघात बहुपद है| अत: दी गई समीकरण एक दिघात समीकरण है|
(c) (x-2)(x+1) = (x-1)(x+3)

=> x2-x-2 = x2+2x-3
=> x2-x-2-x2-2x+3 = 0
=> -3x+1=0

स्पष्ट -3x+1 एक रेखिक बहुपद है| अत: दी गई समीकरण दिघात समीकरण नहीं है|
(d) (x-3)(2x+1) = x(x + 5)

=> x(2x+1)-3(2x+1)-x(x+5) = 0
=> 2x2+x-6x-3-x2-5x = 0
=> x2-10x-3 = 0
स्पष्ट x2-10x-3 एक दिघात बहुपद है| अत: दी गई समीकरण एक दिघात समीकरण है|
(e) (2x-1)(x-3) = (x + 5)(x - 1)

=> (2x-1)(x-3)-(x+5)(x-1) = 0
=> 2x(x-3)-1(x-3)-x(x-1)-5(x-1) = 0
=> 2x2-6x-x+3-x2+x-5x+5 = 0
=> x2-11x+8 = 0
स्पष्ट x2-11x+8 एक दिघात बहुपद है| अत: दी गई समीकरण एक दिघात समीकरण है|
(f) x2+3x+1 = (x-2)2

=> x2+3x+1-(x-2)2 = 0
=> x2+3x+1-(x2-4x+4 ) = 0
=> x2+3x+1- x2+4x-4 = 0
=> 7x-3 = 0
स्पष्ट 7x-3 एक रेखिक बहुपद है| अत: दी गई समीकरण दिघात समीकरण नहीं है|
(g) (x + 2)3 = 2x(x2-1)

=> x3+3x2(2)+3x(2)2+(2)3 = 2x3-2x
=> x36x2+12x+8-2x3+2x = 0
=> -x3+6x2+14x+8 = 0
स्पष्ट -x3+6x2+14x+8 एक त्रिघात बहुपद है| अत: दी गई समीकरण दिघात समीकरण नहीं है|
(h) x3-4x2-x+1 = (x-2)3

=> x3-4x2-x+1 = x3-6x2-12x+8 = 0
=> x3-4x2-x+1-xsup>3+6x2-12x+8 = 0
=> 2x2-13x+9 = 0
स्पष्ट 2x2-13x+9 एक दिघात बहुपद है| अत: दी गई समीकरण एक दिघात समीकरण है|

निम्न स्थितियों को दिघात समीकरण के रूप में निरुपित कीजिये |

(a) एक आयातकर भूखंड का क्षेत्रफल 528 m2 है| क्षेत्र की लम्बाई (मीटरो में ) चौड़ाई के दुगुने से एक अधिक है | हमें भूखंड की लम्बाई और चौड़ाई ज्ञात करनी है|

(b) दो क्रमागत धनात्मक पुर्नानको का गुणनफल 306 है| हमें पूर्णांक को ज्ञात करना है |

(c) रोहन की माँ उससे 26 बर्ष बड़ी है | उसकी आयु (बर्षो में ) का गुणनफल अब से तिन बर्ष पश्चात 360 हो जायेगा| हमें रोहन की बर्तमान आयु ज्ञात करना है

(d) एक रेल गाड़ी 480 km की दुरी समान चाल से तय करता है| यदि चाल 8 km/h कम होती, तो वह उसी दुरी को तय करने में 3 घंटे अधिक लेती है| हमें रेल गाड़ी की चाल ज्ञात करना है|




(a) माना आयातकर प्लाट की लम्बाई और चौड़ाई क्रमश: 2x+1 मीटर और x मीटर है |
इसका क्षेत्रफल = 528 m2 दिया है |
=> 2x2+x-528 = 0
यह वांछित दिघात समीकरण है जो दी गई शर्त को संतुष्ट करता है|
(b) माना दो क्रमागत पूर्णांक x और x + 1 इस प्रकार है की इनका क्षेत्रफल = 306
=> x(x=1) = 360
=> x2+x-306 = 0
\ यह वांछित दिघात समीकरण है जो दी गई शर्त को संतुष्ट करता है|
(c) माना रोहन की वर्तमान आयु x बर्ष है, तो उसकी माता की आयु = (x+3) बर्ष और (x+29) बर्ष ......(1)
माना दिया है की (1) में दर्शाइए आयु का गुणनफल 360 है |
\ अर्थात: (x+3)(x+29) = 360
=> x2+32x87 = 360
=> x2+32x+87-360
=> x2+32x-273 = 0
अत: रोहन की आयु दिघती समीकरण x2+32x-273 को संतुष्ट करती है|
(d) माना x km/h रेल गाड़ी की चाल है|
तो , 480 km में लिया गया समय = 480/x घंटा
480 km में लिया गया समय जब चल 8 km/h कम कर दी जाए = 480/x-8 घंटे
यह दिया है 480 km दुरी तय करने में समय 3 घंटे बड जाता है
=> 480/x-8 - 480/x = 3
=> 480x-480(x-8) = 3x(x-8)
=> 160x-160x+1280 = x2-8x
=> x2-8x-1280 = 0
अत: रेल गाड़ी की चाल दिघातीय समीकरण x2-8x-1280 = 0 को संतुष्ट करती है|
