दिघात समीकरण 4.4




निम्न दिघात समीकरणों के मुलो की प्रकृति ज्ञात कीजिये| यदि मुलो का अस्तित्व हो तो उन्हें ज्ञात कीजिये |

(a) 2x2-3x+5 = 0
(b) 3x2-4√3x+4 = 0
(c) 2x2-6x+3 = 0



Answer :


(a) दी गई समीकरण 2x2-3x+5 = 0
यहाँ , a = 2, बी = -3 और क =5
चूँकि , D = b2-4ac = (-3)2-4x2x5 = 9-40 = -31 < 0
अत: समीकरण के वास्तविक मूल नहीं होगा|
(b) दी गई समीकरण 3x2-4√3x+4 = 0
यहाँ, a = 3, b = -4√3 और c = 4
चूँकि , D = b2-4ac = (-4√3)2-4x3x4 = 48-48 = 0
अत: दी गई समीकरण का वास्तविक मूल होगा, इसका मूल ,
x =-b+-√D/2a = -(-4√3)+-0/2x3 = 2/√3
(c) दी गई समीकरण 2x2-6x+3 = 0
यहाँ, a = 2,b = -6 और c = 3
चूँकि , D = b2-4ac = (-6)2-4x2x3 = 36-24 = 12>0
अत: दी गई समीकरण का वास्तविक मूल होगा, इसका मूल ,
x =-b+-√D/2a = -(-6)+-√12/2x2 = 6+-2√3/4 = 3+-√3/2

निम्न प्रत्येक दिघात समीकरण में k का येसा मान ज्ञात कीजिये की उसके दो बराबर मूल हो |

(a) 2x2+kx+3 = 0
(b) kx(x-2)+6= 0



Answer :


(a) दी गई समीकरण 2x2+kx+3 = 0
यहाँ, a = 2, b = k और k = 0
चूँकि , D = b2-4ac = k2-4x2x3 = k2-24
दी गई समीकरण के वास्तविक और सामान मूल होगे यदि
D = 0, k2-24 = 0
=> k = +-√24 = +-2√6
(b) दी गई समीकरण kx(x-2)+6= 0
यहाँ , a = k, b = -2k और c = 6
चूँकि , D = b2-4ac = (-2k)2-4xkx6 = 4k2-24k
दी गई समीकरण के वास्तविक और सामान मूल होगे यदि
D = 0,
=> 4k2-24k = 0
=> 4k(k-6) = 0
=> k = 0 या k = 6

क्या एक एसी आम की बगिया बनाना संभव है जिसकी लम्बाई , चौड़ाई से दुगुना हो और उसका क्षेत्रफल 800 m2 हो? यदि है , तो उसकी लम्बाई और चौड़ाई ज्ञात कीजिये



Answer :


माना आम की बगिया की लम्बाई 2x और चौड़ाई x है |
क्षेत्रफल = (2x)(x) = 800
=> x2 = 400
=> x = 20
=> x का मान वास्तविक है| अत: बगिया संभव है |
इसकी लम्बाई = 40मी और चौड़ाई = 20 मी

क्या निम्न स्थिति संभव है ? यदि है तो उनकी वर्तमान आयु ज्ञात कीजिये|
दो मित्रो की आयु का योग 20 बर्ष है | चार बर्ष पूर्व उनकी आयु (बर्षो में) का गुणनफल 48 था



Answer :


माना एक मित्रो की आयु = x बर्ष
दुसरे मित्र की आयु = 20x - x
चार बर्ष पहले , पहले मित्र की आयु = (x-4) बर्ष
और , दुसरे मित्र की आयु = (20-x-4) बर्ष = (16 - x ) बर्ष
प्रश्नानुसार, (x-4)(16-x) = 48
=> 16x-x2-64+4x = 48
=> x2-20x+112 = 0
यहाँ , a = 1, b = -20, और c = 112
चूँकि , D = b2-4ac = (-20)2-4x1x112
= 400-448 = -48 < 0
अत: दी गई समीकरण की वास्तविक मूल नहीं है| अत: दी गई स्थिति संभव नहीं है

क्या परिमाप 80 मी तथा क्षेत्रफल 400 m2 के एक पार्क को बनाना संभव है ? यदि है, तो उसकी लम्बाई और चौड़ाई ज्ञात कीजिये |



Answer :


माना लम्बाई x मी और चौड़ाई y मी है|
चूँकि , परिमाप = 80 मी
=> 2(x+y) = 80
=> x+y = 40 ....(1)
साथ ही , क्षेत्रफल = 400 m2
=> xy = 400
=> x(40-x) = 400 [(1) द्वारा]
=> x2-40x+400 = 0
यहाँ , a = 1, b = -40, और c = 400
चूँकि , D = b2-4ac = (-40)2-4x1x400
= 1600-1600 = 0
अत: दी गई समीकरण के वास्तविक मूल है|
इसकी लम्बाई और चौड़ाई निम्न समीकरणों से मिलेंगे |
x2-40+400 = 0
=> (x-20)2 = 0
=> x = 20,20
लम्बाई 20 मी और चौड़ाई 20 मी
पार्क बनाना संभव है |
