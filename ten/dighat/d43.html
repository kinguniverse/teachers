दिघात समीकरण 4.3




यदि निम्नलिखित दिघात समीकरणों के मुलो का अस्तित्व हो तो इन्हें पूर्ण वर्ग बनाने की विधि द्वारा ज्ञात कीजिये :

(a) 2x2-7x+3 = 0
(b) 2x2+x-4 = 0
(c) 4x2+4√3x+3 = 0
(d) 2x2+x+4 = 0



Answer :


(a) 2x2-7x+3 समीकरण x2- 7/2x + 3/2 = 0 के सामान है|
अब , x2- 7/2x + 3/2 = (x - 7/4)2 - (7/4)2+3/2
= (x - 7/4)2 - 49/16+3/2
= (x - 7/4)2 - 25/16
= 2x2-7x+3 = 0
=> (x-7/4)2 - 25/16 = 0
=> (x-7/4)2 = 25/16
=> x - 7/4 = +- 5/4
=> x = 7/4 + 5/4 = 12/4 = 3
और x = 7/4 - 5/4 = 2/4 = 1/2
दी गई समीकरण के मूल 3 और 1/2 होगा|
(b) 2x2+x-4 = 0

=> (x+1/4)2 - (1/4)2-2 = 0
=> (x+1/4)2 - 1/16 -2 = 0
=> (x+1/4)2 - 33/16 = 0
=> (x+1/4)2 = 33/16
=> x+1/4 = +-√33/4
=> x = - 1/4 +-√33/4
चूँकि, x = -1+√33/4
और, x = -1-√33/4
दी गई समीकरण के मूल -1+√33/4 और -1-√33/4 हो|
(c) 4x2+4√3x+3 = 0

=> (2x)2+2x(2x)x√3+(√3)2-(√3)2+3 = 0
=> (2x+√3)2-3+3 = 0
=> (2x+√3)2 = 0
=> x = - √3/2, x = -√3/2
दी गई समीकरण के मूल - √3/2 और - √/2 होगा|
(d) 2x2+x+4 = 0

=> x2+1/2x+2 = 0
=> (x + 1/4)2 - 1/16+2 = 0
=> (x + 1/4)2+31/16 = 0
=> (x + 1/4)2 = - 31/16 < 0
लेकिन , (x + 1/4)2 x के किसी भी वास्तविक मान के ऋणात्मक नहीं हो सकता|
अत: यहाँ x को कोई भी वास्तविक मान दी गई समीकरण को संतुष्ट नहीं करता| अत: दी गई समीकरण का कोई भी वास्तविक मूल नहीं है

उपयुक्त प्रश्न 1 में दिए दिघात समीकरण के मूल, दिघती सूत्र का उपयोग करके, ज्ञात कीजिये|



Answer :


(a) दी गई समीकरण 2x2-7x+3 = 0 है|
यहाँ , a = 2, b = -7 और c = 3
D = b2-4ac = (-7)2-4x2x3 = 49-24 = 25>0
अत: दी गई समीकरण के वास्तविक मूल होगा जो निम्न है :
x = -b +-√D/2a = -(-7) +-√25/2x2 = 7+-5/4
= 12/4 या 2/4 = 3 या 1/2
(b) दी गई समीकरण 2x2+x-4 = 0 है|

यहाँ , a = 2, b=1 और c = -4
चूँकि , D=b2-4ac = (1)2-4x2x-4 = 1+32 = 33>0
अत: दी गई समीकरण के वास्तविक मूल होगा जो निम्न है :
x = -b +-√D/2a = -1 +-√33/2x2 = -1+-√34/4
(c) दी गई समीकरण 4x2+4√3x+3 = 0 है|

यहाँ , a=4, b=4√3 और c=3
चूँकि , D=b2-4ac = (4√3)2-4x4x3 = 48-48 = 0
अत: दी गई समीकरण के मूल वास्तविक और बराबर होगा जो निम्न है
x = -b +-√D/2a = -b/2a = -4√3/2x4 = -√3/2
(d) दी गई समीकरण 2x2+x+4 = 0 है|

यहाँ , a=2, b=1 और c=4
चूँकि , D=b2-4ac = (1)2-4x2x4 = 1 -32 = -31 < 0
अतं: दी गई समीकरण का कोई वास्तविक मूल् नहीं होगा|

निम्न समीकरण का मूल् ज्ञात कीजिये :

(a) x - 1/x = 3, x≠0
(b) 1/x+4 - 1/x-7 = 11/30, x≠-4,7



Answer :


(a) दी गई समीकरण x - 1/x = 3
=> x2-3x-1 = 0
यहाँ , a=1, b=-3 और c=-1
चूँकि , D=b2-4ac = (-3)2-4(1)(-1)=9+4 = 13>0
अत: दी गई समीकरण के वास्तविक मूल होगा जो निम्न है :
x = -b +-√D/2a = -(-3)+-√13/2x1
= 3+-√13/2
(b) दी गई समीकरण 1/x+4 - 1/x-7 = 11/30
=> (x-7)(x+4) / (x+4)(x-7) = 11/30
=> -11/(x+4)(x-7) = 11/30
=> -1/x2-3x-28 = 1/30
=> -30 = x2-3x-28
=> x2-3x+2 = 0
=> (x-1)(x-2) = 0
=> x = 1 या 2
अत: x = 1 और x = 2 दी गई समीकरण के मूल है|

Answer :


माना रहमान की वर्तमान की आयु x बर्ष
प्रश्नुसार , 1/x-3 + 1/x+5 = 1/3
=> 3(x+5)+3(x-3) = (x-3)(x+5)
=> 3x+15+3x-9 = x2+2x-15
=> x2-4x-21 = 0
=> (x-7)(x+3) = 0
=> x =7 या -3
अत: रहमान की वर्मन आयु 7 बर्ष है|

एक क्लास टेस्ट में शेफाली के गणित और अंग्रेजी में प्राप्त किए गए अंको का योग 30 है| यदि उसको गणित में 2 अंक अधिक और अंग्रेजी में 3 अंक कम मिले होते, तो उनके अनके अंको का गुणनफल 210 होता है| उसके द्वारा दोनों विषयों में प्राप्त किये अंक ज्ञात कीजिये |



Answer :


माना शेफाली के गणित में अंक x है| अत: उसके अंग्रेजी अंक (30-x) है|
प्रश्नुसार ,
(x+2)x[(30-x)-3] = 210
=> (x+2)(27-x) = 210
=> 27x-x2+54-2x = 210
=> x2-25x+156 = 0
=> x2-12x-13x+156 = 0
=> x(x-12)-13(x-12) = 0
=> (x-12)(x-13) = 0
=> x = 12 या x = 13
शेफाली के गणित और अंग्रेजी में अंक क्रमश: 12 और 18 है या गणित में और अंग्रेजी में 13 और 17 है|

एक आयताकार खेत का विकर्ण उसकी छोटी भुजा 60 मी अधिक लम्बा है, यदि बड़ी भुजा छोटी भुजा से 30 मी अधिक है, तो खेत की भुजाए ज्ञात कीजिये|



Answer :


माना आयताकार खेत में BC = x मी, तो AC = (x + 60) मी और AB = (x + 30) मी
पाइथागोरस प्रमेय से ,
AC2 = BC2+AB2
=> (x+60)2 = x2+(x+30)2
=> x2+120x+36000 = x2 + x2 + 900
=> x2-60x-2700 = 0
=> x2-90x+30-2700 = 0


=> x(x-90)+30(x-900) = 0
=> (x+30)(x-90) = 0
=> x = -30 या x = 90
चूँकि , x = 90 [आयत की भुजा ऋणात्मक नहीं हो सकती ]
अत: खेत की भुजाए 120 मी और 90 मी है|

Answer :


माना बड़ी संख्या x है तो ,
छोटी संख्या का वर्ग = 8x
और , बड़ी संख्या का वर्ग = x2
संख्या के वर्गों का अंतर 180 दिया है|
चूँकि , x2-8x = 180
=> x2-8x-180 = 0
=> x2-18x+10x-180 = 0
=> x(x-18)+10(x-18) = 0
=> (x-18)(x+10) = 0
=> x-18 = 0 या x+10 = 0
=> x = 18 या x = -10

इस स्थिति में छोटी संख्या का वर्ग = 8x = 8x18 = 144
अत: संख्याए होगी , 18 , 12 या 18 , +-12

स्थिति 2: जब x = -10
इस स्थिति में , छोटी संख्या का वर्ग = 8x = 8x-10 = -80

एक रेलगाड़ी सामान चाल से 360 km की दुरी तय करती है| यदि यह चाल 5 km/h अधिक होती तो वह उसी यात्रा में 1 घंटा कम समय लेती है| रेलगाड़ी की चल ज्ञात कीजिये|



Answer :


माना रेलगाड़ी का एक सामान चाल = x km/h
तो , 360 km तय करने में लिया समय = 360/x घंटा
जब 5 km चाल बड जाती है तो 360 km/h दुरी तय करने में लिया समय = 360/x+5 घंटा
यह दिया है 360 km की दुरी तय करने में 1 घंटा समय कम लगता है |

चूँकि , 360/x - 360/x+5 = 1
=> 360(x+5) - 360x = x(x+5)
=> 360x+1800-360x = x2+5x
=> x2+5x-1800 = 0
=> x2+45x-40x-1800 = 0
=> x(x+45)-40x-1800 = 0
=> x(x+45)-40(x+45) = 0
=> (x-40)(x+45) = 0
=> x-40 = 0 या x+45 = 0
=> x = 40 या x = -45
लेकिन x ऋणात्मक नहीं हो सकता| अत: x = 40
अत: रेलगाड़ी की वास्तविक चाल 40 km/h है|

दो पानी के नल एक साथ एक हौज को 9 x 3/8 घंटो में नल से 10 घंटे कम समय लेते है| प्रत्येक नल द्वारा अलग अलग हौज को भरने का समय ज्ञात कीजिये|



Answer :


माना छोटा नल हौज को भरने में x समय लेता है| अत: बड़ा नल हौज को भरने में (x-10) घंटा लेगा
चूँकि , 1 घंटे में बड़े नल द्वारा भरा हौज का भाग = 1/x-10
=> 9 x 3/8 घंटा अर्थात 75/8 घंटा में बड़े नल द्वारा हौज का भरा भाग
= 1/x-10 x 75/8
इसी प्रकार , 75/8 घंटे में छोटे नल द्वारा भरा भाग = 1/x X 75/8
क्यूंकि यह दिया है की हौज 75/8 घंटे में भरता है |
75/8(x-10) + 75/8x
=> 1/x-10 + 1/x = 8/75
=> x+x-10/x(x-10) = 8/75
=> 75(2x-10) = 8x(x-10)
=> 150x-750 = 8x2-80x
=> 8x2-230x+750 = 0
=> 4x2-115x+375 = 0
=> 4x2-100x-15x+375 = 0
=> 4x(x-25)-15(x-25) = 0
=> (x-25)(4x-15) = 0
=> x-25 = 0 या 4x-15 = 0
=> x = 25 या 15/4
चूँकि , x = 25 क्यूंकि x = 15/4 अमान्य है|
अत: बड़ा नल हौज को 15 घंटे में भरता है और छोटा नल हौज को भरने में 25 घंटे समय लेगा
