In a class test, 3 marks are awarded for each correct answer to the question. And (-2) is given for each wrong answer. And no marks are awarded for not attempting the question

(a) Shivani scored 20 marks. If 12 of his answers are found to be correct. how many questions did he answer wrong



Answer : 8


12 Correct answer to question =12 x 3 = 36
For wrong question = 20 -36 = -16

Marks given for one wrong answer = -2

Number of wrong answer = (-16)/(-2 ) = 8
