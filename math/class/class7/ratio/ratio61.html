There are 45 boys and 35 girls in a class. Find the ratio of the following.

(a) Number of boys to number of girls
(b) Number of girls to number of boys
(c) Number of boys to total number of students




(a) Number of boys to number of girls
number of boys = 45
number of girl = 35

Ratio of number of boys and number of girls
number of boys/number of girls
45/35 = 9/7 = 9:7


(b) Number of girls to number of boys
number of boys = 45
number of girl = 35

Ratio of number of girls and number of boys
number of girls/number of boys
35/45 = 7/9 = 7:9


(c) Number of boys to total number of students
number of boys = 45
number of girl = 35
total number of students = 45+35 = 80

Ratio of number of boys and total number of students
number of boys/total number of students
45/80 = 9/16 = 9:16
